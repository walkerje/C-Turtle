\hypertarget{classcturtle_1_1Turtle}{}\doxysection{cturtle\+::Turtle Class Reference}
\label{classcturtle_1_1Turtle}\index{cturtle::Turtle@{cturtle::Turtle}}


The \mbox{\hyperlink{classcturtle_1_1Turtle}{Turtle}} Class Symbolically represents a turtle that runs around a screen that has a paint brush attached to its tail. The tail can be in two states; up and down. As the turtle moves forwards, backwards, left, and right, it can draw shapes and outlines, write text, and stamp itself onto whatever surface it\textquotesingle{}s walking/crawling on (In this case, it\textquotesingle{}s walking on a Turtle\+Screen).

\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classcturtle_1_1Turtle}{Turtle}} Class Symbolically represents a turtle that runs around a screen that has a paint brush attached to its tail. The tail can be in two states; up and down. As the turtle moves forwards, backwards, left, and right, it can draw shapes and outlines, write text, and stamp itself onto whatever surface it\textquotesingle{}s walking/crawling on (In this case, it\textquotesingle{}s walking on a Turtle\+Screen).
