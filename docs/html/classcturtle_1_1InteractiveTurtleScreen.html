Detailed Description

InteractiveTurtleScreen Holds and maintains facilities in relation to displaying turtles and consuming input events from users through callbacks. This includes holding the actual data for a given scene after being populated by Turtle. It layers draw calls in the order they are called, independent of whatever Turtle object creates it.
